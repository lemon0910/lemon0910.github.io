# 文章头部信息

```
---
layout: post
title:  Markdown CheatSheet
date:   2017-07-29 15:19:00
author: lemon
categories: 其它
---
```

* layout，指定页面样式，固定为post
* title，文章名称
* date，时间，按照 `YYYY-MM-DD HH:MM:SS` 格式书写
* author，作者署名
* categories，文章分类，可以先去首页Category页面看下现有分类，如无合适分类可自行添加
